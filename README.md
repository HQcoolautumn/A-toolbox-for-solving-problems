一个帮助你解决许多烦人事的工具盒，本地html网页，点开即用
有绘画，工具比较全，里面支持ps,sai的笔刷导入，也支持PSD，png等文件导出
有剪辑，里面有AI字幕功能（因为网页上限，所以最好30s一段的识别，不要一次性丢，不然准度不行还容易把网页搞崩溃，当然有本地显卡的选项，不过环境要自己配一下，不太适合小白一键处理）
有调色（类似于达芬奇，适合临时需要调色但是没有下或懒得软件的）
有文档处理，里面有PDF转word,word转PDF，PDF，PPT，word，表格并且都要转图片的能力
有视频图片转换器，可以把小众的格式比如avif转化成大众的png，jpg视频同理（ts格式转换适配有些问题，因为大部分浏览器完全不支持解码，我尽力适配了，但依然可能不行）
有图片编辑，和手机上的差不多，可以剪辑，调色，调锐度，对比度，滤镜啥的
有三维预览（文件不要太大，而且有点小问题）
有翻译（在线本地可选）
注意！！！不同于工程软件，网页没有办法实时保存数据，所以退出数据就没了，记得中途导出保存，尤其是大文件
